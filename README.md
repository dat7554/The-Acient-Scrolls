# The-Acient-Scrolls
A 3D RPG prototype featuring dungeon exploration, narrative progression, and interactive gameplay systems.

## Features 
- **First-Person Movement & Combat:** Navigate environments while engaging enemies with melee attacks, blocking, and stamina-based combat mechanics.
- **Enemy AI:** Enemies patrol, chase, and attack using a finite-state machine with detection based on distance and line-of-sight.
- **Inventory Management:** Collect and organize items in inventory.
- **Interactive World:** Interact with NPCs to trade items and progress the story.
